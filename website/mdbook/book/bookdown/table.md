# Table

Syntax：

```
| Name   | email               |
|--------|---------------------|
| Chung-Chen Chen | <ccckmit@gmail.com> |
| Snoopy | <snoopy@gmail.com>  |
```

Show as：

| Name   | email               |
|--------|---------------------|
| Chung-Chen Chen | <ccckmit@gmail.com> |
| Snoopy | <snoopy@gmail.com>  |

chinese:
# 表格

```
| 姓名   | email               |
|--------|---------------------|
| 陳鍾誠 | <ccckmit@gmail.com> |
| Snoopy | <snoopy@gmail.com>  |
```

呈現結果：

| 姓名   | email               |
|--------|---------------------|
| 陳鍾誠 | <ccckmit@gmail.com> |
| Snoopy | <snoopy@gmail.com>  |
