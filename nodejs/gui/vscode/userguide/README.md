# 編輯技巧

https://code.visualstudio.com/docs/editor/codebasics

## 行選取

Shift and Alt => Column Selection

### Search Across Files

Ctrl+Shift+F 搜尋所有檔案


### 自動排好程式碼

Format Document (Shift+Alt+F) - Format the entire active file.

Format Selection (Ctrl+K Ctrl+F) - Format the selected text.
