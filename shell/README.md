# Shell (bash)

export PS1="--> " 可以讓顯示路徑變短，例如：

```
ccckmit@DESKTOP-96FRN6B:/mnt/d/Dropbox/course/sp106b/linuxc/process$ export PS1="--> "
--> ls
fork  fork.c  pid  pid.c  signal  signal.c  wait  wait.c
```

使用 chsh -s /bin/bash 將 sh 預設為 bash
